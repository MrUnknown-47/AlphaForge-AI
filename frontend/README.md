# AlphaForge Terminal

The institutional portfolio analytics and execution command desk.

## Tech Stack
* **Next.js 15 App Router** for page routing layouts.
* **React 19** for UI render cycles.
* **Zustand** for lightweight state management.
* **TailwindCSS** for styled layouts.
* **Recharts & Framer Motion** for charts.

---

## Directory layout
* `src/app/`: Core route layouts (live-trading, backtesting, portfolio, optimization, copilot, research, hedgefund, operations, agents).
* `src/components/`: Modular widgets.
* `src/store/`: Zustand stores.
* `src/services/`: API injection drivers.

---

## Developer Guide
1. **Install dependencies:**
```bash
npm install
```
2. **Start dev server:**
```bash
npm run dev
```
3. **Compile build check:**
```bash
npm run build
```

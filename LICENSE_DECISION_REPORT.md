# License Decision Report

This report documents the selection of the open-source license for AlphaForge AI.

---

## 1. License Comparison Matrix

| License | Commercial Use | Patent Grant | Trademark Protection | Copyleft Constraints | suitability |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **MIT** | Allowed | No | No | None | High (maximum adoption) |
| **Apache 2.0** | Allowed | Yes | Yes (No trademark transfer) | None | High (institutional standard) |
| **BSD 3-Clause** | Allowed | No | Yes (No name promotion) | None | Medium |
| **GPLv3** | Allowed | Yes | No | Full copyleft | Low (corporate adoption blocker) |

---

## 2. Selected License & Rationale

**Selected License:** **Apache License, Version 2.0 (Apache-2.0)**

### Rationale:
1. **Trademarks Safeguards:** Sections 6 explicitly protects the author's names and trademarks from reuse.
2. **Explicit Patent Grants:** Retains patent litigation clauses to safeguard intellectual property.
3. **Institutional Friendliness:** Broadly accepted by institutional quant hedge funds and enterprise SRE partners.

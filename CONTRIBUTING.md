# Contributing to AlphaForge AI

Welcome! Thank you for contributing.

## Development Workflow

1. **Create Branch:**
```bash
git checkout -b feature/your-feature-name
```
2. **Setup pre-commit hooks:**
Ensure hooks checks are active:
```bash
pre-commit install
```
3. **Format Standards:**
Keep your edits conforming to Black, Isort, and Ruff conventions.
4. **Testing before submit:**
Ensure all pytest assertions pass before initiating PR reviews.

# Security Policy

## Vulnerability Reporting

If you find a security bug, please report it privately:
* Email: security@alphaforge.ai
* Do NOT open public GitHub issues.

## Verification checklist
* Keep secrets out of code.
* Short-lived JWT lifespans (15 mins in prod).
* Enforce position caps (5% max exposure).
